# 説明
M5Stack/M5StickCを利用して、新型コロナウイルス感染症 (COVID-19)の累計が一目で分かります。
いまは、北陸３県に対応しています。

# Hardware
* M5Stack
* M5StickC

# 利用手順
1. M5Stack/M5StickCのいずれかを用意する
1. Githubからソースコードを取得する
1. コード内のSSID/PASSを自環境に合わせて書き換える
1. 取得したい地域のコメントアウトを外す
1. ハードウェアにファームウェアを書き込む
1. 設定完了
